一个基于 Node.js + Express + MySQL + JWT 的社交应用后端服务，提供完整的用户认证、动态发布和个人资料管理功能。

📋 项目概览
技术栈
运行时: Node.js (v18+)
框架: Express.js
数据库: MySQL 8.0+
认证: JWT (JSON Web Token)
工具: nodemon (开发环境)

# 📦 **项目模块划分**

## 🏗️ **三大核心模块**

### 🔐 **1. 认证模块 (Authentication)**
**位置**: `routes/auth.js`  
**API前缀**: `/api/auth`  
**核心功能**:
- ✅ 用户注册（用户名+手机号+密码）
- ✅ 用户登录（支持用户名/手机号双模式）
- ✅ JWT令牌生成与管理
- ✅ 用户退出处理

### 👤 **2. 用户模块 (User)**
**位置**: `routes/user.js`  
**API前缀**: `/api/user`  
**核心功能**:
- ✅ 获取用户个人资料
- ✅ 修改用户密码（多重验证）
- ✅ 用户信息管理

### 📝 **3. 动态模块 (Posts)**
**位置**: `routes/posts.js`  
**API前缀**: `/api/posts`  
**核心功能**:
- ✅ 发布社交动态（需登录）
- ✅ 浏览动态列表（公开）
- ✅ 查看动态详情
- ✅ 更新/删除动态（仅限本人）
- ✅ 完整的CRUD操作

## 🔗 **模块关系**

```
用户流程:
注册(认证模块) → 登录(认证模块) → 获取资料(用户模块) → 发布动态(动态模块)
```

## 🛡️ **权限控制**

- 🔒 **公开接口**: 注册、登录、浏览动态列表
- 🔐 **需登录接口**: 发布动态、修改资料、删除动态
- 👑 **权限验证**: 用户只能操作自己的数据

## 📊 **数据流向**

```
前端请求 → 认证中间件 → 业务模块 → 数据库操作 → 统一响应
```
