# 校园时刻 - Campus Moments

## 🎯 项目简介
一个校园版朋友圈多媒体分享平台，让师生分享校园生活、交流想法。

## 🚀 核心功能
- 👥 用户注册登录
- 📝 多媒体动态发布（图文、视频）
- 💬 评论点赞互动
- 🏷️ 标签分类检索
- 👨‍💼 管理后台

## 🛠️ 技术栈
### 后端
- Node.js + Express
- MySQL
- JWT认证

### 前端  
- UniApp（小程序）
- Vue.js

## 📁 项目结构
campus-moments/
├── backend/ # Node.js后端
├── frontend/ # UniApp前端
├── docs/ # 项目文档
└── database/ # 数据库脚本
